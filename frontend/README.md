# Dalarnia Legends Hub (Next.js)

Интерфейс фан-сайта восстановлен в исходном неон/галактическом стиле: мы перенесли всю прежнюю верстку, классы и анимации в Next.js 14 (App Router) и подключили те же CSS, что использовались во Vite-версии.

## Что внутри

- Next.js 14 + App Router, route handlers (`/api/*`) и статическая генерация страниц.
- Легаси-стили из Vite-проекта (Orbitron + Rajdhani, карточки `.card`, кнопки `.btn`, навигация и т.д.) собраны в `src/styles/legacy.css` и импортируются глобально.
- Mock-данные описаны в `src/data`, доступ к ним дает `src/lib/data-service.ts`, а API-роуты (`/api/articles/[category]`, `/api/legends`, `/api/trading/*`, `/api/community/*`, `/api/decks`) возвращают их в формате, совместимом с прежним фронтом.
- Все страницы (`/articles`, `/legends`, `/decks`, `/builder`, `/community/*`, `/trading/*`) — клиентские компоненты, повторяющие JSX старой SPA-версии, поэтому визуально приложение идентично оригиналу.

## Скрипты

```bash
npm install
npm run dev      # запуск dev-сервера http://localhost:3000
npm run lint     # ESLint
npm run build    # production build
```

## Особенности

- Навигация и футер — прямой порт исходных компонентов; активные ссылки подсвечиваются по тому же правилу, что и раньше.
- Главная страница использует `framer-motion`, как и в старом приложении, поэтому анимация hero/карточек одинакова.
- Билдер колод, страницы сообществ, маркет и покупка токена работают через внутренние API-роуты, так что внешнего backend по‑прежнему не требуется.
- Весь старый Vite-проект сохранён в каталоге `frontend_vite/` на случай, если потребуется сравнение или повторное использование ассетов.
